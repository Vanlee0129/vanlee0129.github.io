注册 NPM 账号

首先去 npm 官网注册账号，注册地址https://www.npmjs.com/。然后验证邮箱即注册完成。

准备要发布的包

准备发布的包在开发好后要注意 package.json 中的如下信息：

  • name 包名
  • version  版本号，更新版本时需要修改版本号
  • dependencies  包依赖

相关目录规范

  • src：源码源文件。
  • lib：依赖文件（没通过 npm，直接下载源码的那种）。
  • node_modules：npm 依赖文件。
  • bin：二进制可执行文件。
  • tests：单元测试或集成测试文件。
  • docs：文档、开发手册。
  • examples：示例代码或项目。
  • build：构建时所需文件。
  • dist：打包后的输出目录。
